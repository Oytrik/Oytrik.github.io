GLMC

Linear Algebra Library for OpenGL in C

This library is meant to be used as a C replacement for glm, with a plan of a wrapper for python.

Work Done

For this project I did some basic mathematical research on the linear algebra being implemented here. I learnt a few of new operations previously unknown to me.

The header functions are coded in C and most of them are hard coded to improve effeciency and reduce operation time. There are various functions that operato upon 2x2, 3x3, 4x4 column major vectors and matrices. Apart from basic arithmetic functions of addition, subtraction, multiplication and division there is also implementation for scaling, rotation, translation of matrices.
